## 📋 Sobre o projeto

Este projeto foi desenvolvido para praticar a criação de sites institucionais voltados para profissionais liberais, com atenção especial à estética, organização das informações e adaptação para diferentes dispositivos.

A proposta foi construir uma experiência acolhedora e profissional, alinhada às necessidades de uma clínica psicológica moderna.

> **Importante:** Todos os dados, imagens e informações apresentados neste projeto são fictícios e utilizados exclusivamente para fins educacionais e de portfólio.

---

## ✨ Funcionalidades

- 🏠 Página inicial com apresentação profissional;
- 👩‍⚕️ Seção "Sobre" com trajetória e abordagem terapêutica;
- 💬 Informações sobre os atendimentos realizados;
- ❓ Perguntas frequentes (FAQ);
- 📱 Botão para contato via WhatsApp;
- 📍 Informações de localização e atendimento;
- 📲 Layout responsivo para celular, tablet e desktop.

## 🛠️ Tecnologias utilizadas

| Tecnologia | Finalidade |
|------------|-------------|
| **TanStack Start v1** | Framework full-stack baseado em React com suporte a SSR/SSG |
| **React 19** | Construção da interface do usuário |
| **TypeScript** | Tipagem estática e maior segurança no desenvolvimento |
| **Vite 7** | Ambiente de desenvolvimento e build |
| **Tailwind CSS v4** | Estilização utilizando classes utilitárias e tokens semânticos |
| **shadcn/ui** | Biblioteca de componentes reutilizáveis |
| **Lucide React** | Conjunto de ícones modernos |
| **Bun** | Gerenciador de pacotes e ambiente de execução JavaScript |
| **Git & GitHub** | Controle de versão e hospedagem do código |
| **Vercel** | Deploy e hospedagem da aplicação |

## ⚙️ Executando o projeto localmente

### Pré-requisitos

- Bun instalado

### Instalação

Clone o repositório:

```bash
git clone https://github.com/EduMPinheiro/psi-mariana-demo.git
```

Entre na pasta do projeto:

```bash
cd psi-mariana-demo
```

Instale as dependências:

```bash
bun install
```

Execute o ambiente de desenvolvimento:

```bash
bun dev
```

